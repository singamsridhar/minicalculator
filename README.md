# 🎌 Anime Club – HTML & CSS Project

## 📌 Project Description

This project is a simple animated website developed using HTML and CSS.
It allows users to explore anime content in a visually attractive interface.

---

## 🚀 Features

* Animated homepage
* Stylish UI design
* Anime cards layout
* Navigation between pages
